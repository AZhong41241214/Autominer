# AutoMiner

適用於 Minecraft 1.21.8 的 Fabric 模組，手持鎬時自動持續挖掘方塊。

目前 Modrinth 上同類型模組最高僅支援至 1.21.7，本模組為目前已知支援 1.21.8 的版本。

## 功能

- 手持任何鎬並對準方塊，自動開始挖掘
- 按下 **R** 鍵切換開啟／關閉
- 相容所有材質的鎬，包含其他模組新增的鎬
- 輕量化設計，無額外設定頁面

## 需求

- Minecraft 1.21.8
- Fabric Loader ≥ 0.16.0
- Fabric API

## 安裝方式

1. 從 [Actions](../../actions) 下載最新的 `.jar`
2. 放入 `.minecraft/mods` 資料夾
3. 使用 Fabric 啟動遊戲

## 快捷鍵

| 按鍵 | 功能 |
|------|------|
| `R` | 切換自動挖掘開啟／關閉 |
